| degree (name)      | polynomial form (general form)                         |
| ------------------ | ------------------------------------------------------ |
| zeroth (constant)  | $f(x)=a_0$ <br /> $f(x)=c$                             |
| first (linear)     | $f(x)=a_{1}x + a_{0}$ <br /> $f(x)=mx+c$               |
| second (quadratic) | $f(x)=a_{2}x^2+ a_{1}x +a_{0}$ <br /> $f(x)=ax^2=bx+c$ |
